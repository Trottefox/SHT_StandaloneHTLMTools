🔒 Hidden Signal — image steganography

Hide an encrypted message inside the pixels of an image, or append it invisibly after the image data. Everything below runs locally in this browser tab — nothing is uploaded anywhere. Open this file with no internet connection and it works exactly the same.

AES-256-GCM SHA-512 key derivation LSB pixel steganography zero external requests
